# Changelog

All notable changes to this project are documented in this file.

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

---

## [1.3.0] — 2026-03-04

### Added

- **Gemini page-by-page sequential comparison** (`ai_page_diffs`) with top-to-bottom ordering and exact text snippets
- **AI Analysis tab** in frontend as default results tab
- **Word-level text highlighting** for changed text blocks
- **Side-by-side overlay ON/OFF toggle** for visual comparison control

### Changed

- **TextDiff schema** now includes `position` for deterministic top-to-bottom ordering
- **Diff overlay pipeline** now uses binarization + alignment + mask cleanup to reduce render-noise false positives
- Updated project documentation (`README.md`, `docs/api.md`, `docs/architecture.md`, `llms.txt`) to match current backend/frontend behavior

---

## [1.2.0] — 2026-03-04

### Changed

- **UI redesign** — Complete frontend overhaul to product-grade app shell with Geist font, dark workspace layout, stats sidebar, and tabbed toolbar
- Replaced all emoji icons with **Lucide SVG icons** for a professional look
- Redesigned diff cards with color-coded badges and expand/collapse
- Side-by-side viewer with dot pagination

### Fixed

- **Similarity percentage bug** — Overall similarity was always 0% because unchanged items were filtered before averaging. Now uses weighted average of all text pair similarity scores (including unchanged)

---

## [1.1.0] — 2026-03-04

### Added

- **Side-by-side visual comparison** with page renders and pixel-level diff overlays
- **Pixel-level diff overlays** — red/green highlights on changed page regions using numpy
- **Gemini Vision** integration for image description and comparison
- **Table semantic analysis** via Gemini AI
- **Overall summary** generated by Gemini with page render context
- **HTML report export** from the frontend
- **Scanned PDF detection** with automatic Tesseract OCR fallback
- **Health check endpoint** (`GET /health`) with OCR and Gemini status
- **Docker Compose** multi-container setup with health checks

### Changed

- Backend Dockerfile now includes `poppler-utils`, `tesseract-ocr`, and `libgl1`
- Frontend Dockerfile uses multi-stage build (deps → build → runner)
- Gemini model name configurable via `GEMINI_MODEL` environment variable
- API key can be provided per-request or via `.env`

---

## [1.0.0] — 2026-03-03

### Added

- Initial release
- **PDF text extraction** with `pdfplumber`
- **Paragraph, heading, and bullet point detection**
- **Table extraction** and cell-level comparison
- **Image extraction** from PDF XObjects via `pypdf`
- **Similarity scoring** with `difflib.SequenceMatcher`
- **Gemini AI** semantic text comparison
- **FastAPI backend** with `/compare` endpoint
- **Next.js frontend** with drag-and-drop upload, tabbed results, and diff viewer
- **Docker support** for both backend and frontend
